# Home Buying Fit Score MVP v11

Version 11 cleans up the Ranked Homes table and improves iPad usability notes.

Changes:
- Compact score/decision badges so "Proceed with Caution" no longer wraps vertically in the ranked table.
- Wider and cleaner ranked table columns.
- Shorter buyer-note preview with line clamping.
- Hash-aware tabs so a tab can be opened directly from a URL fragment such as `#ranked`.
- Added a no-JavaScript warning and iPad note.

Best iPad experience: host the folder as a normal web page and open the URL in Safari or Chrome. Local Files/Quick Look preview may display the page but block JavaScript interactions.
